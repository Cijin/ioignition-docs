# Welcome

Let's discover **ioignition in less than 5 minutes**.

## Getting Started

Get started by **adding a new site**.

Or **try immediately** with **[live demo](https://www.ioignition.com/analytics/ioignition.com)**.

### What you'll need

- Be able to paste the tracking or performance script between the `<head>...</head>` of your page
